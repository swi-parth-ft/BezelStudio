Flexible text styling built for screenshot design

Text layers support fast iteration: adjust fonts, size, weight, alignment, and style while keeping everything editable. This helps you test messaging variants without rebuilding layouts.

  • Multiple font families and typographic styles
  • Per-layer positioning, scale, and rotation
  • Reusable composition patterns across canvases

Visual emphasis without clutter

Gradient fills, stroke, and shadow controls help create contrast when screenshots include bright UI colors or complex backgrounds.

  • Solid and gradient text fills
  • Editable stroke width and color
  • Fine shadow radius and offset controls

Built for App Store readability

Clear text hierarchy is critical for conversion. Bezel Studio helps keep the first message readable while still allowing expressive visual style.
