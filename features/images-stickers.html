From image to sticker in one flow.

Add badges, imported images, and sticker callouts to screenshot layouts. Bezel Studio includes tools for cutouts, border styling, and reusable image presets.

Fast image placement for screenshot callouts

Use badges, logos, and imported images to highlight features in your App Store screenshots. Every image layer can be moved, resized, rotated, and reordered.

  • Photo picker and built-in badge assets
  • Layer-based transform controls
  • Saved image presets for repeated use

Background removal and sticker borders

Convert clean cutouts into sticker-style graphics by adjusting border color and thickness. This is useful for playful callouts and onboarding illustrations.

  • One-tap background removal when supported
  • Custom border width and color
  • Quick reset to original image

Image Playground integration

On supported devices, you can generate sticker graphics from prompts using Apple Image Playground workflows, then insert those outputs directly into your design.
